Thumbnails vs Full-size

Thumbnails: Smaller, optimized images (150×100px). Better for slow internet connections. Automatic generation on-demand.

Full-size: Original image files. Better quality, faster loading on local/fast networks. Recommended for development.

Performance Tips
  • Use thumbnails for remote users
  • Enable lazy loading for large galleries
  • Increase cache timeout for stable configurations
